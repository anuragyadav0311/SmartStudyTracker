# Smart Study Tracker

## Overview

Smart Study Tracker is a Java-based command-line application that helps students track their daily study sessions and improve consistency.

## Features

* Add study sessions (subject, duration, date)
* View all recorded sessions
* Simple and user-friendly interface
* Data stored locally using file handling

## Technologies Used

* Java
* File Handling
* Object-Oriented Programming (OOP)

## Project Structure

```
SmartStudyTracker/
│── src/
│   ├── Main.java
│   ├── StudySession.java
│   ├── TrackerService.java
│── data/
│   ├── sessions.txt
│── README.md
```

## How to Run

1. Clone the repository:

```
git clone https://github.com/anuragyadav0311/SmartStudyTracker.git
```

2. Navigate to project folder:

```
cd SmartStudyTracker/src
```

3. Compile:

```
javac Main.java
```

4. Run:

```
java Main
```

## Example Usage

* Enter subject, duration, and date to log a session
* View all sessions anytime
